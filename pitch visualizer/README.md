 Pitch Visualizer: Text to Storyboard Generator

 Project Description
The Pitch Visualizer is an AI-powered tool that converts narrative text into a visual storyboard. It breaks down a paragraph into scenes, enhances each scene into a descriptive prompt, and generates corresponding images using a text-to-image model.

Working
1. User inputs a paragraph (3–5 sentences)
2. Text is split into scenes using NLTK
3. Each scene is enhanced into a detailed prompt
4. Stable Diffusion generates images
5. Images are displayed as a storyboard in a web interface
